Getting Joobsbox PHP

Joobsbox PHP is simple to get. You simply have to go to http://www.joobsbox.com/download/ in order to get the latest stable version of the software.

Joobsbox PHP development version

In order to get the latest source code available for Joobsbox, go to http://github.com/valentinbora/joobsbox-php/tree/master and hit download or clone the git repo.
